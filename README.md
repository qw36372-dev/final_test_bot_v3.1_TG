# 🧪 ФССП Тест-бот v3.1 TG - Production Ready

Telegram-бот для тестирования сотрудников ФССП по 11 специализациям.

## ✨ Возможности

### Основные функции
- ✅ 11 специализаций тестирования
- ✅ 4 уровня сложности (резерв, базовый, стандартный, продвинутый)
- ✅ Числовые эмодзи кнопки (1️⃣2️⃣3️⃣4️⃣5️⃣)
- ✅ Варианты ответов полностью в тексте сообщения
- ✅ Множественный выбор ответов
- ✅ Таймер обратного отсчета
- ✅ FSM управление состоянием

### Расширенные функции
- 📄 **PDF сертификаты** - автоматическая генерация
- 📊 **Статистика** - история тестов в SQLite
- 🔔 **Напоминания** - раз в неделю неактивным
- 🗑 **Автоудаление** - ответы удаляются через 60 сек
- 🖼 **Логотип** - поддержка изображения
- 🎯 **Команды** - /start, /stats, /help в меню
